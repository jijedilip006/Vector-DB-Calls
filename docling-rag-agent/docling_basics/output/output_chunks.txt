1.4 How to use search records in the employee profile?
With hundreds of employees, it can be an overhead to find a particular entry. A powerful search of JIVIEWS allows you to find an entry in the employee profile based on the following criteria:
1. Employment basis: Searching with the employment basis will list out all the employees within the employee profile
2. Roster group: -Searching with the Roster group will list out all the employees within the employee profile
3. Role group: - Searching with the Role group will list out all the employees within the employee profile.
4. Employment status: Searching with the employment status will list out all the employees within the employee profile.
